## Recent Releases

<Update id="v1-2-0" label="2024-10-15" description="v1.2.0" tags={["feature", "bugfix"]}>

## New Features

- Added real-time collaboration for documents using WebSockets.
- Introduced version history with diff viewer.

## Improvements

- Enhanced search with fuzzy matching and filters.

## Bug Fixes

- Fixed rendering issues with tables containing `{special characters}`.
- Resolved authentication errors when using `YOUR_API_KEY`.

**Example: New collaboration API**

<CodeGroup tabs="JavaScript,Python">
  ```javascript

  collab.onUpdate((changes) => console.log(changes));
  ```
  ```python
  collab = api.start_collaboration('doc-id', user_id='user123')
  collab.on_update(print)
  ```

<Update id="v1-1-0" label="2024-09-20" description="v1.1.0" tags={["improvement", "feature"]}>

## New Features

- Dark mode toggle with automatic system preference detection.
- Export to PDF and Markdown formats.

## Improvements

- Reduced bundle size by 30% through tree-shaking.
- Faster initial load times `<100ms`.

## Breaking Changes

- Updated config format from `{old: value}` to `newConfig.value`.

**Migration Example**

````jsx
<CodeGroup tabs="Before,After">
  ```javascript
  // Before
  config = { apiKey: 'YOUR_API_KEY' };
  ```
  ```javascript
  // After
  config = { credentials: { apiKey: 'YOUR_API_KEY' } };
  ```
</CodeGroup>
````

</Update>
