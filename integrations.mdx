## Overview

Андреев Иван supports seamless integrations with popular third-party services. You can automate workflows, receive notifications, embed content, and build custom solutions using webhooks and the public API. These integrations help you streamline documentation management without leaving your workspace.

Start by exploring supported apps, then configure webhooks for real-time updates. For advanced use cases, embed external content or access the API directly.

  Review your plan limits in the dashboard at `https://dashboard.example.com/account` before enabling premium integrations.
</Callout>

## Setting Up Webhooks

Webhooks notify external services of events like doc publishes or comments. Configure them in your workspace settings.

<Steps>
  <Step title="Navigate to Settings" icon="settings">
    Go to `https://dashboard.example.com/settings/webhooks`.
  </Step>
  <Step title="Add Webhook" icon="plus">
    Click "New Webhook" and enter your endpoint URL, such as `https://your-webhook-url.com/webhook`.
  </Step>
  <Step title="Select Events" icon="zap">
    Choose events like `doc.updated` or `comment.created`.
  </Step>
  <Step title="Test and Save" icon="check-circle">
    Send a test payload and save the configuration.
  </Step>
</Steps>

Verify payloads with this example:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const payload = {
    event: 'doc.updated',
    doc_id: 'doc_123',
    workspace_id: 'ws_456',
    timestamp: Date.now()
  };
  ```
  ```python
  payload = {
      "event": "doc.updated",
      "doc_id": "doc_123",
      "workspace_id": "ws_456",
      "timestamp": 1699123456789
  }
  ```
</CodeGroup>

## Embedding External Content

Embed content from services like YouTube or Figma directly into your docs.

<Tabs>
  <Tab title="YouTube Video" icon="video">
    Use this snippet:

    ````html
    <iframe
      src="https://www.youtube.com/embed/dQw4w9WgXcQ"
      width="560"
      height="315"
      frameborder="0"
      allowfullscreen
    ></iframe>
    ````

    Replace the `src` with your video ID.
  </Tab>
  <Tab title="Figma Design" icon="figma">
    Generate an embed code from Figma:

    ````html
    <iframe
      src="https://www.figma.com/embed?embed_host=share&url=https://www.figma.com/file/abc123"
      width="100%"
      height="600"
    ></iframe>
    ````

    Paste into any doc page.
  </Tab>
</Tabs>

<Callout kind="alert">
  Embeds must comply with `https://dashboard.example.com/policies` to avoid security risks.
</Callout>

## Custom Integrations via API

Build tailored solutions using the REST API at `https://api.example.com/v1`.

<Request tabs="cURL,JavaScript">
  ````bash
  curl -X POST https://api.example.com/v1/webhooks \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{
      "url": "https://your-webhook-url.com/webhook",
      "events": ["doc.updated"]
    }'
  ````
  ````javascript
  const response = await fetch('https://api.example.com/v1/webhooks', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      url: 'https://your-webhook-url.com/webhook',
      events: ['doc.updated']
    })
  });
  ````
</Request>

<Response tabs="200">
  ```json
  {
    "id": "wh_789",
    "url": "https://your-webhook-url.com/webhook",
    "events": ["doc.updated"],
    "active": true
  }
  ```
</Response>

<ParamField path="url" param-type="string" required="true">
  Your webhook endpoint URL.
</ParamField>

<ParamField header="Authorization" param-type="string" required="true">
  Bearer token: `Bearer YOUR_API_KEY`.
</ParamField>

<Expandable title="Advanced API Authentication" default-open="false">

For OAuth, exchange code for tokens:

<CodeGroup tabs="cURL">

  ````bash
  curl -X POST https://api.example.com/v1/oauth/token \
    -d 'grant_type=authorization_code&code=AUTH_CODE&client_id=YOUR_CLIENT_ID'
  ````
</CodeGroup>

</Expandable>
