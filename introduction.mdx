## Overview

Андреев Иван provides a centralized space for your project documentation. You create, organize, and collaborate on docs using an intuitive MDX-based system. Manage everything from API references to user guides in one place, with built-in search, version control, and customizable themes.

<Callout kind="info">
  This documentation space serves as your default hub. Customize it to fit your project's needs.
</Callout>

## Quick Navigation Guide

Get started with these essential steps to navigate and use Андреев Иван effectively.

<Steps>
  <Step title="Browse Sidebar" icon="menu">
    Use the left sidebar to access pages like `/quickstart`, `/authentication`, and `/changelog`. Expand sections for subpages.
  </Step>
  <Step title="Search Documentation">
    Press `<kbd>Ctrl</kbd>+<kbd>K</kbd>` or click the search icon to query across all docs.
  </Step>
  <Step title="Customize Theme">
    Go to settings to adjust the brand color (default: `#3B82F6`) and layout.
  </Step>
  <Step title="Edit Pages">
    Fork or edit MDX files directly in your repo for live previews.
  </Step>
</Steps>

  <Tab title="API Documentation" icon="code">
